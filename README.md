# tigrisek-honlap



## Futtatás

Hozd nyilvánosan a weboldalt a gépeden:

```bash
npm install
npm run dev
```

**Windows-on PowerShell esetén:** Ha "running scripts is disabled" hibaüzenetet kapsz, futtasd ezt először:

```powershell
Set-ExecutionPolicy RemoteSigned -Scope CurrentUser -Force
```

Ezután nyisd meg a böngészőben: http://localhost:5173

A fejlesztési szerver a `src/main.tsx`-ből indul.
